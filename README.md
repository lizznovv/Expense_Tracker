# Expense Tracker

Web application for managing personal expenses and monthly budgets.

The system allows users to track expenses, categorize transactions, monitor spending, and visualize financial statistics.

---

## Features

- Add, edit, and delete expenses
- Expense categories and filtering
- Monthly budget management
- Budget limit warnings
- Expense statistics and charts
- CSV export support
- Sorting and date filtering

---

## Categories

- Food
- Transport
- Entertainment
- Utilities
- Shopping
- Other

---

## Technologies

- Laravel
- PHP 8+
- MySQL
- Blade / Vue.js
- Chart.js

---

## Validation Rules

- Expense amount must be positive
- Category must be valid
- Date must use correct format
